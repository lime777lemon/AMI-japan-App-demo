## 🚀 Vercelでデプロイする手順

### ステップ1: Vercelにアクセス
https://vercel.com にアクセス

### ステップ2: ログイン
- "Sign Up" または "Log In" をクリック
- **GitHubアカウントでログイン**（lime777lemon）

### ステップ3: プロジェクトを追加
1. "Add New Project" をクリック
2. リポジトリ一覧から **"AMI-japan-App-demo"** を選択
3. "Import" をクリック

### ステップ4: 設定を確認
以下の設定が自動検出されます（そのままでOK）:
- **Framework Preset**: Vite
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Install Command**: `npm install`

### ステップ5: デプロイ
- "Deploy" ボタンをクリック
- 2-3分待つと完了！

### ステップ6: デモURLを取得
デプロイ完了後、以下のようなURLが表示されます：
- **本番URL**: `https://ami-japan-app-demo.vercel.app`
- または: `https://ami-japan-app-demo-lime777lemon.vercel.app`

このURLを投資家に共有できます！
